ðŸ“š Color Formats Explained

HEX (Hexadecimal)

Six-digit code representing RGB values in hexadecimal (base-16). Format: #RRGGBB

Example: #3498db = Red: 34, Green: 98, Blue: db (219 in decimal)

Use in CSS: color: #3498db;

RGB (Red, Green, Blue)

Additive color model using values 0-255 for each color channel.

Example: rgb(52, 152, 219)

Use in CSS: background: rgb(52, 152, 219);

With alpha: rgba(52, 152, 219, 0.5);

HSL (Hue, Saturation, Lightness)

More intuitive for humans. Hue (0-360Â°), Saturation (0-100%), Lightness (0-100%).

Example: hsl(204, 70%, 53%)

Use in CSS: color: hsl(204, 70%, 53%);

HSV (Hue, Saturation, Value)

Similar to HSL but with Value instead of Lightness. Common in color pickers.

Example: hsv(204, 76%, 86%)

Note: Not directly supported in CSS, but useful for color manipulation

CMYK (Cyan, Magenta, Yellow, Key/Black)

Subtractive color model used in printing. Values are percentages.

Example: cmyk(76%, 31%, 0%, 14%)

Note: Used for print design, not web development

ðŸŽ¨ Color Theory Tips:

  • 60-30-10 Rule: Use 60% dominant color, 30% secondary, 10% accent
  • Contrast: Ensure text has sufficient contrast for readability (WCAG AA: 4.5:1)
  • Complementary: Great for call-to-action buttons and highlights
  • Analogous: Perfect for creating serene, comfortable designs
  • Triadic: Provides visual contrast while maintaining harmony
