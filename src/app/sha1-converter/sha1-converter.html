# From string
$text = "your text here"
$sha1 = [System.Security.Cryptography.SHA1]::Create()
$hash = $sha1.ComputeHash([System.Text.Encoding]::UTF8.GetBytes($text))
[System.BitConverter]::ToString($hash) -replace '-',''

hash_object = hashlib.sha1(text.encode())

ğŸ“š About SHA1 (Secure Hash Algorithm 1)

SHA-1 is a cryptographic hash function designed by the United States National Security Agency (NSA) and published in 1995. It produces a 160-bit (20-byte) hash value, typically rendered as a 40-digit hexadecimal number.

How SHA1 Works:

  1. Padding: Message is padded to multiple of 512 bits
  2. Processing: Message divided into 512-bit blocks
  3. Compression: Each block processed through 80 rounds
  4. Output: Final 160-bit hash value produced

SHA-1 is no longer considered secure for cryptographic purposes. Theoretical attacks were demonstrated in 2005, and practical collision attacks became feasible in 2017.

Do NOT use SHA-1 for:

  • Password hashing
  • Digital signatures
  • SSL/TLS certificates
  • Security-critical applications
